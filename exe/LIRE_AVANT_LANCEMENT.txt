MEDISCAN AI - EXECUTABLE LOCAL

Ce dossier contient uniquement les fichiers utiles pour lancer MediScan AI.

Le projet complet est dans le dossier parent :
  mediscan/

Executable principal macOS / Linux :
  Double-cliquer sur MEDISCAN_EXECUTABLE.command

Executable Windows :
  Double-cliquer sur MEDISCAN_EXECUTABLE_WINDOWS.bat

Si Windows a deja echoue pendant l'installation des dependances :
  1. Relancer MEDISCAN_EXECUTABLE_WINDOWS.bat
  2. Si l'erreur continue, supprimer le dossier .venv311 dans mediscan/
  3. Relancer MEDISCAN_EXECUTABLE_WINDOWS.bat

Pourquoi deux fichiers ?
  Windows utilise les fichiers .bat.
  macOS et Linux utilisent les scripts shell, ici fournis en .command pour le double-clic.

Pre-requis :
  - Python 3.11
  - Node.js 20.19+ ou 22.12+
  - Git LFS si le projet est reclone depuis Git

Une fois lance, l'application est disponible ici :
  Frontend : http://127.0.0.1:5173
  Backend  : http://127.0.0.1:8000

Note :
  Les fichiers FAISS du dossier artifacts/ sont geres par Git LFS.
  Dans ce clone, git lfs pull a deja ete execute.
